Tokocrypto CLI

Trade, track markets, and manage your account from the terminal. Built-in MCP server for AI agent integration.

View on GitHub Install via NPM
$ tokocrypto ticker btc/idr
# High-performance market data...

$ tokocrypto mcp -s all
# MCP server active — ready for Claude, Cursor, and Gemini

🚀 Quick Start

npm install -g @ibidathoillah/tokocrypto-cli
tokocrypto ticker btc/idr

🤖 AI Agent Integration

Built-in Model Context Protocol (MCP) support. Connect your exchange account to AI agents securely.

{
  "mcpServers": {
    "tokocrypto": {
      "command": "tokocrypto",
      "args": ["mcp", "-s", "all"]
    }
  }
}
